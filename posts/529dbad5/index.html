Go字符串长度、比较与转义符、格式化输出、strings包下的方法

发表于2023-08-30|更新于2023-08-30|Go
|字数总计:1k|阅读时长:4分钟|阅读量:

字符串长度、比较与转义符、格式化输出、strings包下的方法

字符串长度、比较与转义符

len()

  • len(): 内置方法 获取字符串长度
    1. 英文字母不论大小写，每个字母都占1个字节
    2. 每个中文字都占3个字节
    3. 字符串是由字符组成的 [I, \s, l, o, v, e, \s, y, o, u]；这些字母或空格都可以看成一个 byte或rune
    4. 中文的情况下要用rune转切片

转义符

  • \: 将符号意义转换为普通文本字符的特性
  • \\: 将转义符转义
  • \r: 回车
  • \n: 换行
  • \r\n: 搭配一起用，换行
  • \t: Tab键，制表符

字符串的比较

  • 相等：==

  • 不等：!=

  • 条件：是字符串是否顺序与每个字母都相等

  • 比较大小时：字符串依次进行ASCII码对比

格式化输出

Println, Print

  • Println => print line
    • P => 包外使用的方法
    • fmt => Println => 包外使用
    • 有回车换行 \r\n

Printf, Sprintf

  • Printf => print format

  • Sprintf => 将格式化的字符串返回

strings.Builder

性能最高的

  • strings => 字符串方法包
    • Builder => WriteString()

字符串的一些方法

strings是一个包

  • strings.Contains()包含、检查（检查字符串中是否包含子字符串内容）
    • 参数：
      1. 字符串
      2. 子字符串
    • 返回值：bool

  • strings.Count("我正在学习Go", "Go")检查字符串中出现子字符串的次数

  • 参数：

    1. 字符串
    2. 子字符串

  • 返回值：int

  • strings.Split() 分割字符串组成切片 []

    • 参数：
      1. 字符串
      2. 用什么进行分割的
    • 返回值：切片
  • strings.Index() 子字符串在字符串中的位数
    • 参数：
      1. 字符串
      2. 子字符串
    • 返回值：int
  • strings.Replace() 替换字符串中的某个子字符串
    • 参数：
      1. 字符串
      2. 需要替换的字符串
      3. 新字符串
      4. 替换几个(-1是全部替换)
    • 返回值：string
  • strings.Trim() 去除字符串前后指定的字符
    • 参数：
      1. 字符串
      2. 指定要去掉前后的什么字符
    • 返回值：string

  • strings.ToLower() 将所有字符串转为小写

    • 参数：str
    • 返回值：string

  • strings.ToUpper() 将所有字符串转为大写

    • 参数：str
    • 返回值：string

  • strings.HasPrefix() 是否有某个前缀字符串

    • 参数：
      1. str
      2. 前缀字符串
    • 返回值：string

  • strings.HasSuffix() 是否有某个后缀字符串

    • 参数：
      1. str
      2. 前缀字符串
    • 返回值：string
