4. 变量声明

  • Variable Declaration
  • JavaScript：ES3 var
  • JavaScript：ES6 let const
    • ES6：面向对象推给JS开发者

一、JavaScript 4种声明变量的方法

  1. var ES3 不推荐
  2. implicit 暗示全局变量 ES3 不推荐
  3. let ES6
  4. const ES6

2、let

3、const

二、变量初始化

  • 变量第一次被赋值的过程

    1. 声明一个变量
    2. 将某一个值赋值给一个变量 （var let 可选，暗示 const 必选）
    3. 开发者手动进行赋值
    4. 系统默认进行赋值 undefined

  • 声明变量必定会有给该变量进行初始化值的过程

三、单一声明

四、分号书写规则

  • JS引擎会分析代码自动加上必要的分号
  • ES3-5：推荐打分号（约定俗成的规范）
  • ES6：没有推荐
  • 注意：
    • 要打都打，要不打都不打
    • 立即执行函数建议在前面打分号
    • 语句不打分号

五、var

  • 全局作用域变量（globally-scoped Variable全局变量）
  • 函数作用域变量（function-scoped Variable函数局部变量）

1、什么是作用域

  • 什么是作用域：限定变量可访问权限的范围

2、全局作用域

  • 全局作用域：基于window或者globalThis的可访问范围

    1. 一个HTML的访问，会产生一个对应着该HTML的JavaScript全局作用域。任何脚本被该HTML引入，就共享一个全局作用域
    2. 全局作用域变量：在该全局作用域直接声明的变量。在全局作用域声明的变量，系统会在全局对象（window、globalThis）中创建对应的属性
    3. 之所以可以不用全局对象访问变量，是因为全局作用域属性对所有其他

  • 总结：在全局作用域直接用var声明的变量，是以全局对象属性的形式被创建的

3、函数作用域

  • 函数作用域：function body（函数体）的访问范围
    1. 函数内部的作用域叫函数作用域，也叫函数局部作用域
    2. 局部作用域内可访问全局作用域的变量
    3. 全局作用域无法访问函数作用域的变量
    4. 函数可以嵌套函数，作用域可以嵌套作用域
  • 块级作用域：基于block的访问范围

4、变量提升

  • Variable Hoisting

    1. 代码在执行的最开始，会进行预编译操作
    2. 预编译的过程
      1. 【预编译阶段】找到var -> 变量a
      2. 【预编译阶段】放入预编译容器 { a }
      3. 【预编译阶段】初始化 { a: undefined }
      4. 【执行期阶段】打印a -> undefined
      5. 【执行期阶段】a = 1 将1赋值给变量a

  • 总结：尽量按照程序的逻辑顺序进行变量声明和访问；变量提升var声明的一个缺点（BUG）

5、暗示全局变量（暗示implicit）

  1. 未用任何变量声明关键字段变量
  2. 暗示全局变量必须手动进行变量初始化
  3. 暗示全局变量默认为全局变量

6、var最不合理的几点

  1. 可以重复声明
  2. 变量提升 （声明之前可以访问）
  3. 暗示全局变量 （可以不用声明变量的关键字声明变量）

六、let

1、概念

  • ES6 block 块
    • block-scoped 块级作用域
    • 类似于函数作用域
  • 限制变量可访问范围的工具作用域：块
  • 这个范围就是块作用域
  • 在ES6中
    • 函数体
    • if
    • for
    • while
    • 或直接用 {} 声明的块等包含的{}都是块作用域

2、块作用域可以单独进行创建 {}

  • let声明：声明块作用域的局部变量

3、let声明变量的访问限制

4、块作用域与函数作用域

  1. var声明的变量，访问只能被限制在函数作用域范围内
  2. let声明的变量，访问可以被限制在块作用域和函数作用域内

5、重复声明

  • let声明不允许重复声明

6、let跟var的冲突

  • 在同一作用域下，相同标识符不可以同时被var跟let声明

七、const

  • const -> constant
  • con + stant
  • come 一起，全部
  • stant：stand 站立
  • constant -> 状态不变
  • const：常量（常数变量）不可变的量
    • const的特征与let几乎一致

1、const与let不同点

  1. 必须手动初始化：const a = 1;
  2. 在初始化后，不可改变其值

2、书写

  • 常量一般使用全大写，单词之前用下划线分隔
  1. 所有不能变更的值或者值不变的量都应该使用const
  2. 只有后续需要变更值的量使用let
  3. var尽量不适用
